# 英语学习助手 V2

一个基于间隔重复算法的智能英语学习桌面应用，帮助你在摸鱼时间高效学习英语单词。

## 🎯 项目特色

- **智能间隔重复**：基于科学的学习算法，自动安排复习时间
- **摸鱼学习模式**：后台运行，定时弹窗提醒，不影响正常工作
- **强制学习机制**：答错题目无法关闭，必须答对才能继续
- **雅思高频词汇**：内置50个雅思高频词汇，首次启动自动加载
- **精美界面设计**：采用Material Design风格，界面简洁美观
- **跨平台支持**：支持Windows、macOS、Linux系统

## 🚀 核心功能

### 📚 生词本管理
- 添加、编辑、删除单词
- 支持中文解释和例句
- 智能搜索功能（单词/含义/例句）
- 熟练度系统（0-5星评级）

### 🧠 智能学习系统
- **间隔重复算法**：根据熟练度自动调整复习间隔
- **费曼学习法**：熟练度5星后采用长期复习策略
- **学习统计**：详细的学习进度和正确率统计

### 🎮 摸鱼学习模式
- **定时提醒**：自定义学习间隔，定时弹出单词测试
- **强制学习**：答错题目时窗口无法关闭，必须答对
- **系统托盘**：启动学习模式后自动最小化到系统托盘
- **后台运行**：不影响正常工作，静默学习

### ⚙️ 个性化设置
- 学习间隔时间自定义
- 强制学习模式开关
- 开机自启动设置
- 声音提醒开关
- 系统托盘设置

## 🛠️ 技术栈

- **核心框架**: Electron 25.0.0
- **前端框架**: Vue 3 + Vite
- **状态管理**: Pinia
- **数据库**: SQLite3
- **定时任务**: Node Schedule
- **UI设计**: Material Design + 自定义样式
- **打包工具**: electron-builder

## 🎮 使用指南

### 首次使用
1. 启动应用后，点击"初始化雅思词汇"按钮
2. 系统会自动加载50个雅思高频词汇
3. 点击"开始学习"进入摸鱼学习模式

### 添加单词
1. 在生词本页面点击"添加单词"按钮
2. 填写英文单词、中文含义和例句（可选）
3. 点击"添加"保存

### 学习模式
1. 点击顶部"开始学习"按钮
2. 应用会自动最小化到系统托盘
3. 根据设定的时间间隔，系统会弹出单词测试窗口
4. 选择正确的中文含义，答对后窗口自动关闭
5. 答错时窗口保持显示，必须答对才能继续

### 查看统计
- 在"统计"页面查看学习进度
- 包括总单词数、已掌握数量、复习次数、正确率等

### 个性化设置
- 在"设置"页面调整学习参数
- 可以修改学习间隔、开启强制模式等

## 📊 学习算法

### 间隔重复算法
- **0星**：30分钟后复习
- **1星**：1小时后复习
- **2星**：4小时后复习
- **3星**：1天后复习
- **4星**：3天后复习
- **5星**：1周后复习

### 费曼学习法
- 熟练度达到5星后，采用2周间隔复习
- 当天已学习过的单词不再重复弹出

## 🔧 开发说明

### 项目结构
```
learnEnglishV2/
├── src/
│   ├── main/                 # Electron 主进程
│   │   ├── index.js         # 主进程入口
│   │   ├── database.js      # SQLite 数据库管理
│   │   ├── scheduler.js     # 定时任务调度
│   │   └── preload.js       # 预加载脚本
│   ├── renderer/            # 渲染进程 (Vue 3)
│   │   ├── components/      # Vue 组件
│   │   ├── views/          # 页面视图
│   │   ├── store/          # Pinia 状态管理
│   │   └── main.js         # Vue 应用入口
│   └── shared/             # 共享代码
│       ├── constants.js    # 常量定义
│       └── utils.js        # 工具函数
├── data/                   # 数据文件
│   └── ielts-words.json    # 雅思高频词汇
├── tests/                  # 测试和调试文件
│   ├── README.md          # 测试文件说明
│   ├── debug-*.js         # 调试脚本
│   ├── test-*.js          # 测试脚本
│   └── check-*.js         # 数据库检查脚本
└── dist/                   # 构建输出
```

### 核心模块

#### 数据库管理 (database.js)
- SQLite数据库操作
- 单词CRUD操作
- 学习记录管理
- 设置保存

#### 学习调度器 (scheduler.js)
- 定时任务管理
- 间隔重复算法实现
- 学习模式控制

#### 状态管理
- **wordStore**: 单词数据管理
- **studyStore**: 学习状态管理
